urlstd#

urlstd is a Python implementation of the WHATWG URL Living Standard.

This library provides URL class, URLSearchParams class, and low-level APIs that comply with the URL specification.

Supported APIs#

  • URL class: urlstd.parse.URL

    • canParse: can_parse()

    • href: href

    • origin: origin

    • protocol: protocol

    • username: username

    • password: password

    • host: host

    • hostname: hostname

    • port: port

    • pathname: pathname

    • search: search

    • searchParams: search_params

    • hash: hash

  • URLSearchParams class: urlstd.parse.URLSearchParams

    • append: append()

    • delete: delete()

    • get: get()

    • getAll: get_all()

    • has: has()

    • set: set()

    • sort: sort()

  • Low-level APIs

    • URL parser: urlstd.parse.parse_url()

    • basic URL parser: urlstd.parse.BasicURLParser.parse()

    • URL record: urlstd.parse.URLRecord

      • scheme: scheme

      • username: username

      • password: password

      • host: host

      • port: port

      • path: path

      • query: query

      • fragment: fragment

      • origin: origin

      • is special: is_special()

      • is not special: is_not_special()

      • includes credentials: includes_credentials()

      • has an opaque path: has_opaque_path()

      • cannot have a username/password/port: cannot_have_username_password_port()

      • URL serializer: serialize_url()

      • host serializer: serialize_host()

      • URL path serializer: serialize_path()

    • Hosts (domains and IP addresses)

      • IDNA

        • domain to ASCII: urlstd.parse.IDNA.domain_to_ascii()

        • domain to Unicode: urlstd.parse.IDNA.domain_to_unicode()

      • host parser: urlstd.parse.Host.parse()

      • host serializer: urlstd.parse.Host.serialize()

    • percent-decode a string: urlstd.parse.string_percent_decode()

    • percent-encode after encoding: urlstd.parse.string_percent_encode()

    • application/x-www-form-urlencoded parser: urlstd.parse.parse_qsl()

    • application/x-www-form-urlencoded serializer: urlstd.parse.urlencode()

    • Validation

      • Hosts (domains and IP addresses): urlstd.parse.HostValidator

        • valid host string: is_valid()

        • valid domain string: is_valid_domain()

        • valid IPv4-address string: is_valid_ipv4_address()

        • valid IPv6-address string: is_valid_ipv6_address()

      • URL: urlstd.parse.URLValidator

        • valid URL string: is_valid()

        • valid URL-scheme string: is_valid_url_scheme()
